## ğŸ¯ About The Project

**VoxCheck AI** is an intelligent voice authentication system designed to detect whether an audio clip is generated by AI or spoken by a real human. In an era where AI-generated voices are becoming increasingly sophisticated and being used for deepfakes, scams, and misinformation, VoxCheck AI provides a critical layer of verification and security.

### ğŸ” Why VoxCheck AI?

With the rise of advanced AI voice synthesis technologies like ElevenLabs, OpenAI's TTS, and others, distinguishing between real and synthetic voices has become nearly impossible for the human ear. VoxCheck AI leverages machine learning algorithms to analyze audio patterns and provide accurate detection results.

## ğŸš€ Getting Started

### Prerequisites

Ensure you have the following installed:

- **Node.js** (v16 or higher)
- **npm** (v8 or higher)
- **Git**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/rahulsvt-1907/guvi-hackathon.git
   cd guvi-hackathon
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   
   Navigate to `http://localhost:5173` to see the application running.

### Build for Production

```bash
npm run build
```

The optimized production build will be generated in the `dist/` directory.

## ğŸ’» Usage

### Basic Usage

1. **Upload Audio File**
   - Click on the upload area or drag and drop your audio file
   - Supported formats: MP3, WAV, M4A, OGG

2. **Wait for Analysis**
   - The system will automatically process your audio
   - Analysis typically completes in 2-5 seconds

3. **View Results**

- **Media Analysis**: Authenticate voice recordings in journalism and research
- **Personal Safety**: Verify voice messages from unknown sources
